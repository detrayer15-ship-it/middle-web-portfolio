# Портфолио Данияла для CapEducation

Современный сайт-портфолио «Обо мне» для конкурса CapEducation в направлении Middle Web.

## Разделы сайта

1. О себе.
2. Моя цель.
3. Как я пришел в IT.
4. Мой ментор.
5. Точка А → Точка Б.
6. Хобби и интересы.
7. Мои лучшие работы.
8. Ссылка на GitHub.

## Технологии

- HTML5.
- CSS3.
- JavaScript.
- Flexbox и Grid.
- Адаптивная верстка через медиазапросы.
- Fetch API для загрузки данных из `data.json`.
- Анимации появления разделов при прокрутке.
- Фиксированное меню разделов с анимированной шестерней.
- Бургер-меню для телефона.

## Как добавить и просмотреть MP4-работы

У вас есть два удобных способа воспроизвести видео-презентации на сайте:

1. **Автоматически (размещение файлов в проекте)**:
   Создайте папку `assets/videos/` в корне проекта и сохраните туда ваши файлы под именами:
   - `assets/videos/html-css-js-practice.mp4` (для практического урока HTML/CSS/JS)
   - `assets/videos/tilda-practice.mp4` (для практического урока Tilda)
   После этого видео автоматически будут загружаться и воспроизводиться в соответствующих карточках. При отправке проекта на GitHub просто добавьте эти файлы в коммит.

## Фотография

Сейчас используется локальная запасная аватарка. Для настоящей фотографии добавьте файл
`assets/images/profile.jpg` и замените `profile.svg` на `profile.jpg` в `index.html`.



## Файлы проекта

- `index.html` - структура сайта.
- `style.css` - стили, адаптивность и CSS-анимации.
- `script.js` - меню, Fetch API, анимации и прогресс.
- `data.json` - факты, прогресс, хобби и работы.
- `assets/images/favicon.svg` - иконка вкладки браузера.
- `README.md` - описание проекта.
